# Sticky Navbar Demo

This small demo implements a sticky navigation bar with hover and scroll effects and a responsive mobile menu.

Files added:
- [index.html](index.html)
- [styles.css](styles.css)
- [script.js](script.js)

To test locally, run a simple static server in the project root and open http://localhost:8000/index.html

Example using Python 3:
```bash
python3 -m http.server 8000
```
